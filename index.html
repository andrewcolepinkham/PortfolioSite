Lights Out

React Project

Lights Out is a fun grid-based puzzle game that I built using React. The UI includes that hint feature which implements a custom linear algebra algorithm I developed originally in Java and then migrated to JS. The algorithm utilizes a series of binary gaussian eliminations performed on n-dimensional linear equations and by proof with solve the puzzle correctly every time. Light Wall

React | Flask | Raspberry Pi

I built a 100-channel Raspberry Pi relay board to progmatically control 100 individual lighbulbs on a wall-mounted grid. Via live video-feed and website, users can toggle the state of each lightbulb in real-time by drawing over the website's live-feed video-image. A REST-ful Flask API controls the light board based on real-time UI input. A 1080p Raspberry Pi camera model sends video-feed data to the React UI so users can see the feed.
